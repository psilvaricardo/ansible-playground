# Ansible Playground

A hands-on Ansible playground for practicing automation patterns, building reusable roles, and exploring real-world infrastructure workflows. This repository combines practical DevOps automation with guided examples, annotated playbooks, and module experimentation—ideal for learning, testing, and mastering Ansible from fundamentals to advanced usage.

---

## 🚀 Features

- Structured playbook examples (beginner → advanced)  
- Reusable roles with best-practice directory layout  
- Inventory samples for multiple environments (dev/stage/prod)  
- Jinja2 templating demonstrations  
- Ad-hoc command examples  
- Module deep-dives (file, user, service, package, copy, template, etc.)  
- Error-handling, handlers, loops, conditionals, and tags  
- CI/CD friendly structure  
- Real-world automation scenarios for Linux hosts
